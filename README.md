# 经济规律
## 经济规律
| 货币政策 | 措施 | 货币 |
| - | - | - |
| 紧缩 | 提高利率、减少货币供应 | 升值 |
| 宽松 | 降低利率、增加货币供应 | 贬值 |

* 利率引发的变化

| 利率 | 货币 | 债券 | 股票 |
| - | - | - | - |
| 上升 | 升值 | 下跌 | 下跌 |
| 降低 | 贬值 | 上涨 | 上涨 |

## 股市规律
> 进入下跌周期原因：**经济衰退，利率上升，股市泡沫(如纳指100的PE到40)，重大事件(新冠病毒/大型战争)**

* 通货膨胀：实物资产和资源类资产能较好抵御贬值风险
* 通货紧缩：买债券(降息会推动债券价格上涨)和必需行业
* 利率上升：买银行和保险股
* 利率下降：买成长股和科技股
