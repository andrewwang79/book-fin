# 技术分析工具
## MA(移动平均线)
| 项 | 原理 | 使用场景 |
| - | - | - |
| 简单移动平均线(SMA) | 均值 | 低波动市场（债券、外汇），识别稳定的中长期趋势 <br/> 中长期SMA(如 50日、100日)常被市场视为共识性的支撑位或阻力位 |
| 指数移动平均线(EMA) | 近期价格赋予更高权重，远期价格权重逐渐递减 | 高波动市场，适合短线交易中判断趋势的快速变化 |

## K线图
* 实体(body) : 开盘价和收盘价的情况，绿是下跌(顶是开盘价)，红是上涨(顶是收盘价)
* 影线(Wick/Shadows) : 实体上下的线段，分别表示最高价和最低价。
* K线的改变(涨跌)看红顶和绿底
* 5日线/200日线等都是移动平均线

## MACD
* 指数平滑异同移动平均线（Moving Average Convergence Divergence）。[MACD指标的内在逻辑](https://www.zhihu.com/question/29954111)
* 构成
    * DIF线(差离值) = 快速EMA - 慢速EMA。一般是(EMA12 - EMA26)，源自美股交易周期（12≈1个月交易日，26≈2个月交易日）
    * DEA线(信号线，Differential Exponential Average)是DIF线的EMA9。源自约2周交易日
    * Histogram(柱状图) = DIF - DEA。正值时柱状图为红色/空心柱，负值为绿色/实心柱
* 指标应用
    * 金叉：DIF上穿DEA，视为买入信号（柱状图由负转正）
    * 死叉：DIF下穿DEA，视为卖出信号（柱状图由正转负）
    * [底背离](https://xueqiu.com/7021965736/224675817)：股价创新低，但柱状图或DIF未创新低，预示反弹。通过指标验证 “下跌动能衰竭”。“股价跌、指标强” 的背离状态，暗示空方力量减弱、多方力量潜伏，股价可能即将反转上涨
    * 顶背离：股价创新高，但柱状图或DIF未创新高，预示回调
    * 柱状图由长变短收缩，表明趋势减弱
    * 柱状图由短变长扩张，表明趋势加强

## VWAP(成交量加权平均价格)
* 衡量特定时间段内的平均交易价格，常用于**日内交易和短期分析**，帮助交易者判断当前价格相对于平均水平的强弱。
* 计算方式是通过将每个交易价格乘以相应的成交量，然后将这些乘积求和并除以总成交量。$$VWAP = \frac{\sum (\text{价格} \times \text{成交量})}{\sum \text{成交量}}$$
* 市场强弱判断
    * 价格在VWAP上方：市场通常被视为强势，多方力量占优
    * 价格在VWAP下方：市场可能处于弱势，空方力量较强

## BOLL(布林线)
* 通过股价与轨道的位置关系，判断市场波动率、趋势强度及潜在反转信号。核心是“波动率 + 趋势”，通过轨道形态判断市场情绪和多空力量
* 由三条线构成：中轨（通常为20日简单移动平均线，SMA）、上轨（中轨 + 2倍标准差）、下轨（中轨 - 2倍标准差）。

### 场景与对应策略
| 场景分类 | 技术特征描述 | 市场含义解读 | 对应交易策略 | 风险提示 |
| - | - | - | - | - |
| **布林带收窄（收缩）** | 上轨、中轨、下轨距离逐渐缩小，带宽（上轨-下轨）达到近期低点 | 市场波动性降低，多空双方力量暂时平衡，即将迎来方向选择（突破行情前兆） | 1. 等待突破信号确认，不提前入场<br>2. 突破上轨时做多，突破下轨时做空<br>3. 突破后设置止损（如跌破中轨或反方向突破点） | 警惕假突破，需结合成交量放大确认趋势强度 |
| **布林带上轨突破** | 价格突破上轨，且通常伴随成交量放大，中轨向上倾斜 | 多头力量较强，市场进入强势上涨阶段，可能延续趋势 | 1. 顺势做多，入场点为突破上轨时<br>2. 止损设置在下轨或近期低点下方<br>3. 若价格回踩中轨支撑有效，可加仓 | 避免追涨过高，若价格跌破中轨且无法收回，需止盈离场 |
| **布林带下轨突破** | 价格跌破下轨，成交量放大，中轨向下倾斜 | 空头力量占优，市场处于下跌趋势，可能延续调整 | 1. 顺势做空，入场点为跌破下轨时<br>2. 止损设置在上轨或近期高点上方<br>3. 若价格反弹中轨压力有效，可加仓 | 下跌趋势中谨慎抄底，若价格突破中轨且站稳，需止损离场 |
| **布林带中轨震荡** | 价格在中轨附近上下波动，上轨下轨平行或小幅倾斜，带宽变化不大 | 市场处于横盘整理阶段，无明显趋势，多空博弈激烈 | 1. 上轨附近做空，下轨附近做多<br>2. 止损设置在轨道外侧（上轨上方/下轨下方）<br>3. 波动幅度较小时减少仓位 | 突破方向不明确时避免重仓，若突破轨道需及时调整策略（跟随趋势） |
| **布林带上轨缩口回落** | 价格触及上轨后，带宽缩小，价格开始回落，中轨走平或向下 | 多头力量减弱，上涨动能不足，可能转为调整或下跌 | 1. 多头持仓止盈离场<br>2. 若回落中轨支撑失效，可反手做空<br>3. 观察下轨支撑力度，决定是否再次做多 | 高位缩口回落风险较高，避免抱有“继续上涨”的侥幸心理 |
| **布林带下轨缩口反弹** | 价格触及下轨后，带宽缩小，价格开始反弹，中轨走平或向上 | 空头力量减弱，下跌动能衰竭，可能转为反弹或上涨 | 1. 空头持仓止盈离场<br>2. 若反弹中轨压力突破，可反手做多<br>3. 观察上轨压力力度，决定是否再次做空 | 低位缩口反弹需确认有效性，避免过早抄底导致止损 |
| **布林带喇叭口扩张** | 上轨向上快速发散，下轨向下快速发散，带宽急剧扩大，中轨方向明确（向上/向下） | 市场趋势加速形成，波动剧烈，趋势动能强劲 | 1. 中轨向上时坚定做多，止损下轨下方<br>2. 中轨向下时坚定做空，止损上轨上方<br>3. 趋势未改变前不轻易逆势操作 | 喇叭口扩张时趋势较强，但波动大，需严格设置止损，避免趋势反转时大幅亏损 |
| **价格沿上轨运行** | 价格持续在上轨附近或上方运行，中轨陡峭向上，下轨同步上行 | 市场处于极强趋势，多头力量主导，短期超买但趋势未改 | 1. 持仓做多，不轻易止盈<br>2. 止损上移至中轨或下轨附近<br>3. 若价格突然跌破上轨且成交量放大，警惕回调 | 极强趋势中允许一定回调，但需关注中轨支撑，跌破中轨且无法收回时必须止盈 |
| **价格沿下轨运行** | 价格持续在下轨附近或下方运行，中轨陡峭向下，上轨同步下行 | 市场处于极弱趋势，空头力量主导，短期超卖但趋势未改 | 1. 持仓做空，不轻易止盈<br>2. 止损上移至中轨或上轨附近<br>3. 若价格突然突破下轨且成交量放大，警惕反弹 | 极弱趋势中避免过早平仓，但若价格突破中轨且站稳，需及时止损 |

### 使用核心原则与避坑点
1. 不单独依赖：BOLL线需结合成交量、MACD、RSI等指标确认信号，避免单一指标误导（如无量突破多为假突破）。
1. 适配趋势周期：日线BOLL适合中期趋势，小时线BOLL适合短期波段，不同周期信号可能冲突，需明确操作周期。
